ZeroX AI Agent

Your Gemini-powered AI code agent for VS Code

Overview

ZeroX is a powerful VS Code extension that brings the intelligence of Google's Gemini models directly into your editor. It's designed to help you code faster, write better code, and learn more effectively.

Features

  • AI-powered chat for coding questions and assistance
  • Code generation and refactoring
  • Error analysis and debugging help
  • Seamless integration with your VS Code workflow

Getting Started

To get started with ZeroX, you'll need to install the extension from the VS Code Marketplace and configure your Gemini API key.

  1. Install the extension from the VS Code Marketplace (link to be added).
  2. Open the settings in VS Code (File > Preferences > Settings).
  3. Search for "ZeroX" and enter your Google Gemini API key.
  4. Start chatting with ZeroX through the sidebar!
